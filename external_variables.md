# External-validation variables

The external-validation file is a separately processed lactation-level file with the same required core structure as the development file:

- `cow_id` or mapped `Dam ID`;
- `parity` or mapped calving parity;
- `milk_1` to `milk_9`;
- `dim_1` to `dim_9`.

External analyses use core predictors only. External farm identity and exact geographic coordinates are restricted.
